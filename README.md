## 模仿《Philosophical Transactions》的 MiKTeX 模板

这是一个基于 **KOMA-Script** 的 LaTeX 模板，用于以 _Philosophical Transactions_（17 世纪英国皇家学会出版物）的风格排版文章，并已配置为能够在 **MiKTeX** 和 **pdfLaTeX** 环境下顺畅运行。

> 本项目基于以下项目 fork 而来：[https://www.overleaf.com/project/61270a1e3aa10416153cbfdb](https://www.overleaf.com/project/61270a1e3aa10416153cbfdb)，感谢原作者的贡献。

## A MiKTeX Template for Phil. Trans.

A KOMA-based LaTeX template for typesetting articles in the style of _Philosophical Transactions_ (17th-century Royal Society publications), configured to work smoothly with **MiKTeX** and **pdfLaTeX**.

> ACKNOLEDGEMENT - This project was forked from [https://www.overleaf.com/project/61270a1e3aa10416153cbfdb](https://www.overleaf.com/project/61270a1e3aa10416153cbfdb), thanks for their contribution.
